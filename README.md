### 1. Supabase Backend (Required for Multiplayer)
Since this app runs in the browser, it needs a cloud database to sync players.
1. Create a free project at [supabase.com](https://supabase.com).
2. Go to the **SQL Editor** and run:
   ```sql
   create table rooms (
     id bigint generated by default as identity primary key,
     code text unique not null,
     created_at timestamp with time zone default timezone('utc'::text, now()) not null,
     host_id text,
     status text default 'LOBBY',
     settings jsonb default '{}'::jsonb,
     game_state jsonb default '{}'::jsonb,
     players jsonb default '[]'::jsonb,
     updated_at timestamp with time zone default timezone('utc'::text, now()) not null
   );
   alter table rooms enable row level security;
   create policy "Public rooms access" on rooms for all using (true);
   create policy "Public rooms update" on rooms for all using (true) with check (true);
   create policy "Public rooms insert" on rooms for all with check (true);
   ```
3. Open `app.js` and replace the keys at the top:
   ```javascript
   const SUPABASE_URL = 'YOUR_PROJECT_URL';
   const SUPABASE_KEY = 'YOUR_ANON_KEY';
   ```

### 2. Run Locally
- You can just open `index.html` in your browser.
- Or use a simple server like `python -m http.server 8000`.
